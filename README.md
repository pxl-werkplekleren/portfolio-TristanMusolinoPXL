# Online Portfolio - Werkplekleren 1

Dit is mijn persoonlijke online portfolio voor het vak **Werkplekleren 1** binnen de opleiding **Graduaat Programmeren**. 

In dit portfolio documenteer ik mijn groei, competenties en projecten gedurende het eerste semester van mijn opleiding.

## Over Mij
* **Naam:** Tristan Musolino
* **Opleiding:** Graduaat in het programmeren 
* **Focus:** Softwareontwikkeling (C#, SQL, Web) & Design

## Project Overzicht
Dit portfolio is gebouwd als een single-page applicatie (SPA) met een focus op een strakke, professionele weergave van mijn leerproces. 

### Inhoud:
- **Logboek:** Een gedetailleerd overzicht van de wekelijkse activiteiten en evaluaties.
- **Ontwikkeling:** Reflectie op mijn keuze voor de opleiding en situering binnen de professionele rollen.
- **Competenties:** Koppeling aan de Opleidingsspecifieke Leerresultaten (OLR's).
- **Opdrachten:** Overzicht van uitgevoerde projecten zoals Scratch, C# en WPF.
- **Reflectie:** AIM-reflectie op mijn groeiproces en de X-factor waarden.

## Gebruikte Technologieën
* **HTML:** Voor de structurele opbouw van de content.
* **CSS:** Voor de styling, inclusief een responsive grid-layout en PXL-branding.
* **Flexbox & CSS Grid:** Voor de positionering van elementen en de X-factor blokken.
* **Git/GitHub:** Voor versiebeheer en hosting.
